Meta Model — What We Built

Framework: PyMC-Marketing (Bayesian MMM)

The model learns how spend in each channel translates to 1-year cohort LTV, accounting for:

  • Carryover effects — ads influence behavior days after initial exposure
  • Diminishing returns — doubling spend does not double customers
  • Seasonality — the tax-season lift (January–April) is modeled explicitly

What we’ve done:

  • Model specified for Meta iOS, Android, and Web (3 channels × 603 days)
  • Model structure validated — prior predictive checks confirm the expected shapes
  • Full model training runs this week once data questions are resolved

Three Questions to Finalize the Full Model

1. LTV file update New file shows uniformly higher LTV values. Same 1-year definition, or updated LTV model? Determines: how we label ROAS outputs

2. Decision timeline When do you next use budget allocation outputs for an actual spend decision? Determines: precision bar — directional vs. committed spend confidence

3. Channel continuity Any new channels planned for Q2/Q3 2026? Is CTV still active after the Oct–Nov test? Determines: model scope for the planning horizon
